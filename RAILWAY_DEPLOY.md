# PLAGENOR 4.0 - Railway Deployment Guide

## Step 1: Login to Railway
Open your terminal and run:
```bash
railway login
```
This will open a browser window. Complete the login/signup process.

## Step 2: Initialize Project
In your project directory:
```bash
cd c:/Users/hp/Desktop/App/plagenor_django/PLAGENOR_4.0_DJANGO
railway init
```
Select "Empty Project" and name it "plagenor-4"

## Step 3: Add PostgreSQL Database
```bash
railway add postgresql
```

## Step 4: Deploy Application
```bash
railway up
```

## Step 5: Set Environment Variables
```bash
# Generate a secure secret key
railway variables set SECRET_KEY="$(openssl rand -base64 50)"

# Set other required variables
railway variables set DEBUG="False"
railway variables set ALLOWED_HOSTS="*"
railway variables set CSRF_TRUSTED_ORIGINS="https://*.up.railway.app"

# Get the database URL (auto-generated by Railway)
railway variables get DATABASE_URL
```

## Step 6: Run Migrations
```bash
railway run python manage.py migrate
```

## Step 7: Create Superuser
```bash
railway run python manage.py createsuperuser
```

## Step 8: Collect Static Files
```bash
railway run python manage.py collectstatic --noinput
```

## Step 9: Open Your App
```bash
railway open
```

## Troubleshooting

### View Logs
```bash
railway logs
```

### Check Status
```bash
railway status
```

### Redeploy
```bash
railway up
```
